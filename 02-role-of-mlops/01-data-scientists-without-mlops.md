# Role of a Data Scientist

👉 A Data Scientist is a person who:

> takes raw data → understands it → builds a model → stops there

### 1. Understand the Problem

👉 First, they ask:

- What are we solving?
- What should the model do?

Example:

“Predict if a user will cancel subscription”

### 2. Look at the Data

👉 They open data like:

- Excel
- CSV
- Database

They check:

- What columns are there
- Missing values
- Data types

👉 Just understanding the data

### 3. Clean the Data

👉 Fix problems in data:

- Remove duplicates
- Fill missing values
- Correct wrong data

👉 Real-world data is messy, so this step is important

### 4. Explore the Data (EDA)

👉 They try to find patterns:

- Who cancels more?
- Any relationships?

👉 Helps decide what will work

### 5. Build a Model

👉 Now they use ML:

- Choose simple algorithm
- Train model using data

👉 Model learns patterns

### 6. Check the Model

👉 They test:

- Accuracy
- Mistakes

👉 Ask:

“Is this model good enough?”

### 7. Save the Model

👉 Save as file:

- .pkl

👉 Share with team

### 🚨 Important Point

👉 After this:

- Model is just a file
- Not deployed
- Not used by users

### 🔥 One-Line Understanding

👉

> Data Scientist = builds and saves the model, but does not run it in real world
